# Change: Alpha Release - First Public Release

## Why

The alpha release marks Syntlas's first public availability. This phase focuses on release engineering, documentation, community setup, and establishing processes for ongoing development.

## What Changes

- Create release notes and changelog
- Build binary releases for Linux/macOS/Windows
- Write installation documentation
- Create CONTRIBUTING.md guidelines
- Provide example tome structure
- Set up GitHub repository (issues, PRs, wiki)
- Create website/landing page (optional)
- Establish community channels (Discord, etc.)
- Define security disclosure policy

## Impact

- Affected specs: `release`
- Affected code: CI/CD, documentation
- Dependencies: Phase 8 (Testing) must be complete
