# Math Module

Provides access to mathematical functions defined by the C standard.

## Constants

```python
import math
print(math.pi)
print(math.e)
```

## Functions

```python
math.sqrt(16)       # 4.0
math.ceil(4.2)      # 5
math.floor(4.8)     # 4
math.pow(2, 3)      # 8.0
math.log(100, 10)   # 2.0
```
