# Conditional Statements

Python uses indentation to define blocks of code that execute conditionally.

## Basic if Statement

```python
age = 18

if age >= 18:
    print("You are an adult")
```

## if-else Statement

```python
age = 15

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

## if-elif-else Statement

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(grade)  # "B"
```

## Multiple Conditions

```python
age = 25
has_license = True

# and operator (both must be True)
if age >= 18 and has_license:
    print("Can drive")

# or operator (at least one must be True)
if age < 18 or not has_license:
    print("Cannot drive alone")

# not operator
if not has_license:
    print("Need to get license")
```

## Nested Conditions

```python
age = 25
has_ticket = True

if age >= 18:
    if has_ticket:
        print("Can enter venue")
    else:
        print("Need ticket")
else:
    print("Too young")
```

## Truthiness in Python

Values are considered True or False in boolean context.

```python
# Falsy values (evaluate to False)
if False: pass
if None: pass
if 0: pass
if 0.0: pass
if "": pass
if []: pass
if {}: pass
if set(): pass

# Truthy values (evaluate to True)
if True: pass
if 1: pass
if -1: pass
if "hello": pass
if [1]: pass
if {"a": 1}: pass
```

## Ternary Operator (Conditional Expression)

```python
# Syntax: value_if_true if condition else value_if_false
age = 20
status = "adult" if age >= 18 else "minor"
# "adult"

# Use in expressions
max_val = a if a > b else b
```

## Chained Comparisons

```python
age = 25

# Python allows chaining
if 18 <= age <= 65:
    print("Working age")

# Equivalent to
if age >= 18 and age <= 65:
    print("Working age")
```

## Short-Circuit Evaluation

```python
# and: stops at first False
if False and expensive_function():
    pass  # expensive_function() never called

# or: stops at first True
if True or expensive_function():
    pass  # expensive_function() never called
```

## Common Patterns

### Default value handling
```python
value = input_value if input_value is not None else "default"
```

### Validation
```python
def process_data(data):
    if not data:
        return None

    if not isinstance(data, dict):
        raise ValueError("Expected dict")

    # Process data
    return data
```

### Guard clauses
```python
def divide(a, b):
    # Handle edge cases early
    if b == 0:
        return None
    if a == 0:
        return 0

    # Main logic
    return a / b
```

> **Note**: Python doesn't have a switch/case statement. Use if-elif chains or match/case (Python 3.10+).
