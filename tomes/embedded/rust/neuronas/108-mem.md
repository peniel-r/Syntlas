# Memory Operations

Utilities for manipulating memory and ownership.

## swap

Swaps the values of two mutable references.

```rust
use std::mem;

let mut x = 5;
let mut y = 10;

mem::swap(&mut x, &mut y);
assert_eq!(x, 10);
assert_eq!(y, 5);
```

## take

Replaces a mutable reference with its default value, returning the old value. Useful for moving out of struct fields.

```rust
use std::mem;

let mut v = vec![1, 2];
let old_v = mem::take(&mut v); // v is now empty Vec
```
